# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Installed packages:

-   tailwind css(with config)
-   shadcn(with config)
-   react router dom
-   react hot toast
-   moment js
-   react icons
-   react spinners
-   prop types
-   firebase
-   axios
